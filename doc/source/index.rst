Welcome to Python-nxs's documentation!
======================================

The :py:mod:`python-nxs` package provides two bindings to the NeXus C-API

* :py:mod:`nxs.napi` which is only a very simple wrapper create by means of the
  :py:mod:`ctypes` module
* :py:mod:`nxs.tree` which provides a higher level interface to the C-API
